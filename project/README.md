# Recipe Book with Voice Guidance

A modern recipe management app with text-to-speech capabilities, customizable voice selection, and background music.

## Features

- 🎤 **Voice Selection** - Choose from available system voices
- 🎵 **Background Music** - Optional music while reading recipes
- 👍👎 **Recipe Ratings** - Thumbs up/down for each recipe
- 🌓 **Dark/Light Theme** - Toggle between themes
- 🔊 **Text-to-Speech** - Listen to recipe steps
- ⏱️ **Adjustable Delay** - Control pause between steps
- 🔐 **User Authentication** - Secure login system
- 📱 **Responsive Design** - Works on all devices

## How to Get a Real Link (Deploy Online)

### Option 1: GitHub Pages (Free & Easy)

1. Create a GitHub account at https://github.com
2. Create a new repository (e.g., "recipe-book")
3. Upload these files: `index.html`, `app.js`, `styles.css`
4. Go to Settings → Pages
5. Select "main" branch and save
6. Your link will be: `https://[your-username].github.io/recipe-book`

### Option 2: Netlify (Free & Fast)

1. Go to https://www.netlify.com
2. Sign up for free
3. Drag and drop your project folder
4. Get instant link: `https://[random-name].netlify.app`
5. Optional: Customize your domain name

### Option 3: Vercel (Free)

1. Go to https://vercel.com
2. Sign up with GitHub
3. Import your repository
4. Deploy automatically
5. Get link: `https://[project-name].vercel.app`

### Option 4: Render (Free)

1. Go to https://render.com
2. Create a new Static Site
3. Connect your GitHub repository
4. Deploy and get your link

## Quick Deploy Steps (Netlify - Recommended)

1. Go to https://app.netlify.com/drop
2. Drag the entire project folder onto the page
3. Wait 30 seconds
4. Get your live link!
